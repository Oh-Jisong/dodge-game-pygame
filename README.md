# Dodge Game (Pygame)

Python과 Pygame으로 만든 간단한 닷지(Dodge) 게임입니다.

플레이어를 좌우로 움직여 위에서 떨어지는 장애물을 피하고
가능한 한 오래 살아남는 것이 목표입니다.

> 이 프로젝트는 **Pygame을 배우며 만든 Version 1** 입니다.

---

## 게임 특징
- 좌 / 우 이동
- 위에서 떨어지는 장애물
- 생존 시간 기반 점수
- 게임 오버 화면 및 재시작 기능

---

## 조작 방법
- **← / → 방향키** : 이동
- **R** : 게임 오버 후 재시작
- **ESC** : 게임 종료

---

## 실행 환경
- Python 3.9 이상 권장
- Pygame

---

## 설치 방법
```bash
pip install pygame

---

## 실행
python dodge_game.py
